# Session Summary — BayazidxMarin

## Achieved
- **Unified server**: `main.py` on port 5050 serves both Bayazid and Marin; `app.py` removed, `marin_chat.html` deleted
- **Agent switching**: `/agent/switch`, `/chat?agent=`, URL bar indicator, prompt-only swap (no model reload)
- **Marin memory clear restricted**: `/memory/clear` returns "cannot be cleared" when `ACTIVE_AGENT == "marin"`
- **Audio stop**: `pkill -f piper-tts` via `/audio/stop` endpoint + Stop button in UI (hidden on mobile)
- **Max tokens**: configurable via `/settings/maxtokens`, slider in UI
- **RAG toggle**: `/settings/rag` endpoint, toggle button in UI
- **Word limit**: `/settings/wordlimit`, distinct from max_tokens (limits output words vs truncates input)
- **Arena**: `/arena` on port 5070 with live streaming debate, judge mode, history viewer
- **All links fixed**: header nav, arena link, chat load, timers, profile
- **Quiz**: interactive MCQ via `/quiz/generate/json` → parsed from LLM markdown → clickable options with correct/incorrect highlighting + explanations. Timer context injected for both agents.
- **RAG server**: runs with `--max-memory-mb 800` hard limit (aggressive purge + SIGHUP restart)
- **RAG strict**: `/rag/query` returns 503 if server is down; `/settings/rag` always reflects actual server status
- **Button disable on streaming**: true for quiz, teach, debug, etc.

## Key Facts
- `main.py` replaces old `app.py`; only `uvicorn main.py` runs
- Marin loads from `marin.py`, Bayazid from `bayazid.py`; both imported in `main.py`
- Arena runs as separate `uvicorn arena:app` on port 5070
- `templates/bayazid_chat.html` is the shared chat UI (was combined from old `bayazid_chat.html` + `marin_chat.html`)
- File watch pattern: `uvicorn main.py --reload` + `uvicorn arena:app --reload --port 5070` in parallel terminals
